# 🤖 Telegram Group Moderator Bot

Готовый бот для автоматической модерации Telegram-групп. Идеальное решение для владельцев крупных сообществ.

## ✨ Возможности

- 🛡 **Антиспам** — автоматическое удаление спама
- 🔗 **Блокировка ссылок** — запрет на публикацию ссылок
- 🤬 **Фильтр мата** — удаление нецензурной лексики
- 👋 **Приветствие новых** — настраиваемое приветствие
- ✅ **Капча для новых** — защита от ботов (опционально)
- 📊 **Статистика** — отслеживание активности
- ⚙️ **Гибкие настройки** — под любую группу
- 👮 **Команды модерации** — бан, мут, предупреждения
